# SDK Overview

LLM Cost Ops provides official SDKs for multiple programming languages.

## Available SDKs

### Python
Best for data science, ML workflows, and backend applications.

```bash
pip install llm-cost-ops
```

[Learn more →](./python)

### TypeScript
Best for Node.js backends and browser applications.

```bash
npm install @llm-cost-ops/sdk
```

[Learn more →](./typescript)

### Rust
Best for systems programming and maximum performance.

```bash
cargo add llm-cost-ops
```

[Learn more →](./rust)

### Go
Best for high-performance backend services.

```bash
go get github.com/llm-devops/llm-cost-ops-go
```

[Learn more →](./go)

## Common Features

All SDKs provide:

- Cost calculation
- Usage tracking
- Analytics and forecasting
- Report generation
- Async/await support
- Type safety
- Comprehensive error handling
