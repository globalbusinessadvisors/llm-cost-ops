# Deployment Overview

Deploy LLM Cost Ops to your infrastructure with confidence.

## Deployment Options

### Docker
Run LLM Cost Ops in Docker containers for quick development and testing.

### Kubernetes
Deploy to Kubernetes with our comprehensive Helm charts for production workloads.

### Cloud Platforms
- AWS ECS/EKS
- Google Cloud Run/GKE
- Azure Container Instances/AKS

## Infrastructure Requirements

- Database: PostgreSQL 14+ or SQLite
- Memory: 512MB minimum, 2GB recommended
- CPU: 1 core minimum, 2+ cores recommended
- Storage: 10GB minimum for logs and database

## Security Considerations

- Use TLS/SSL for all connections
- Configure RBAC appropriately
- Enable audit logging
- Set up network policies
- Use secrets management
