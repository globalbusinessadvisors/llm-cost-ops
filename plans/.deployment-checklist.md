# Documentation Deployment Checklist

Use this checklist to ensure proper deployment setup.

## Initial Setup

### Local Development
- [ ] Install Node.js 20 or higher
- [ ] Clone repository
- [ ] Navigate to website directory: `cd website`
- [ ] Install dependencies: `npm install`
- [ ] Test local build: `npm run dev`
- [ ] Verify site loads at http://localhost:3000

### Environment Configuration
- [ ] Copy `.env.example` to `.env.local`
- [ ] Configure environment variables (if needed)
- [ ] Test production build: `npm run build`
- [ ] Test production serve: `npm run serve`

### Code Quality
- [ ] Run type checking: `npm run typecheck`
- [ ] Run linter: `npm run lint`
- [ ] Check formatting: `npm run format:check`
- [ ] Verify all checks pass

## GitHub Pages Setup

### Repository Configuration
- [ ] Go to repository Settings
- [ ] Navigate to Pages section
- [ ] Set Source to "GitHub Actions"
- [ ] Save changes

### Custom Domain (Optional)
- [ ] Add CNAME record in DNS:
  - Type: CNAME
  - Name: docs
  - Value: `<username>.github.io`
- [ ] Add custom domain in GitHub Pages settings: `docs.llm-cost-ops.dev`
- [ ] Enable "Enforce HTTPS"
- [ ] Wait for DNS propagation (up to 24 hours)

### Verify Deployment
- [ ] Push to main branch
- [ ] Check GitHub Actions tab for workflow run
- [ ] Verify workflow completes successfully
- [ ] Access deployed site at configured URL
- [ ] Test all navigation and links

## Netlify Setup (Optional)

### Account Setup
- [ ] Sign up at netlify.com
- [ ] Connect GitHub account
- [ ] Import repository

### Site Configuration
- [ ] Configure build settings:
  - Base directory: `website`
  - Build command: `npm run build`
  - Publish directory: `website/build`
  - Node version: 20
- [ ] Save settings

### Environment Variables
- [ ] Add required environment variables in Netlify dashboard
- [ ] Test build in Netlify UI

### GitHub Integration
- [ ] Get Netlify Site ID from Site Settings
- [ ] Get Netlify Auth Token from User Settings
- [ ] Add GitHub secrets:
  ```bash
  gh secret set NETLIFY_AUTH_TOKEN
  gh secret set NETLIFY_SITE_ID
  ```

### Custom Domain
- [ ] Go to Domain Management in Netlify
- [ ] Add custom domain: `docs.llm-cost-ops.dev`
- [ ] Configure DNS as instructed
- [ ] Enable HTTPS (automatic)
- [ ] Verify domain works

### Verify Deployment
- [ ] Push to main branch
- [ ] Check Netlify dashboard for deployment
- [ ] Verify deployment succeeds
- [ ] Check preview deployments work for PRs
- [ ] Test custom domain
