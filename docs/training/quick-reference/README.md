## How to Use These Materials

### For New Team Members

**Week 1:**
1. Start with [Onboarding Checklist](onboarding-checklist.md)
2. Follow Day 1 and Week 1 sections
3. Reference [Quick Reference Card](quick-reference-card.md) for commands
4. Use [Glossary](glossary.md) for unfamiliar terms

**Month 1:**
5. Continue with [Onboarding Checklist](onboarding-checklist.md)
6. Study [Cheat Sheet](cheat-sheet.md) for your language
7. Practice with code examples
8. Complete role-specific sections

### For Developers

**Daily Use:**
- Keep [Quick Reference Card](quick-reference-card.md) handy
- Reference [Cheat Sheet](cheat-sheet.md) for SDK usage
- Check [Glossary](glossary.md) for terminology

**Integration:**
- Start with SDK quick start in [Cheat Sheet](cheat-sheet.md)
- Copy code examples for your language
- Follow best practices section
- Reference API endpoints in [Quick Reference Card](quick-reference-card.md)

### For Administrators

**Setup:**
- Use [Migration Guide](migration-guide.md) for initial setup
- Follow admin sections in [Onboarding Checklist](onboarding-checklist.md)
- Reference configuration in [Quick Reference Card](quick-reference-card.md)

**Operations:**
- Use [Quick Reference Card](quick-reference-card.md) for daily commands
- Check [Cheat Sheet](cheat-sheet.md) for security checklist
- Reference troubleshooting sections

### For Analysts

**Getting Started:**
- Review analyst section in [Onboarding Checklist](onboarding-checklist.md)
- Learn query commands from [Quick Reference Card](quick-reference-card.md)
- Study export options in [Cheat Sheet](cheat-sheet.md)

**Daily Work:**
- Reference query examples
- Use export commands
- Check glossary for metrics terminology

### For Managers

**Onboarding:**
- Use manager section in [Onboarding Checklist](onboarding-checklist.md)
- Review team onboarding process
- Understand success criteria

**Operations:**
- Monitor team progress with checklists
- Reference support contacts
- Review security and compliance sections

## Quick Links

### Most Common Tasks

**Track Usage:**
```python
# Python
client.usage.create(
    provider="openai",
    model="gpt-4-turbo",
    prompt_tokens=1000,
    completion_tokens=500
)
```

**Query Costs:**
```bash
cost-ops query --range last-7-days --output table
```

**Generate Report:**
```bash
cost-ops summary --period last-30-days
```

**Export Data:**
```bash
cost-ops export --output costs.csv --format csv
```
