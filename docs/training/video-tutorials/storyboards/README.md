# Video Tutorial Storyboards

## Overview

This directory contains detailed text-based storyboards for key LLM Cost Ops video tutorials. Storyboards provide visual direction for video production, describing shot composition, screen elements, transitions, animations, and timing.

## Purpose of Storyboards

Storyboards serve as blueprints for video production:
- **Pre-production Planning**: Visualize the video before recording
- **Production Guidance**: Direct camera angles, screen captures, and visual elements
- **Team Coordination**: Align instructors, editors, and animators
- **Quality Control**: Ensure consistent visual language across videos
- **Efficiency**: Reduce production time with clear direction

## Storyboard Format

Each storyboard follows this structure:

### Scene Information
- **Scene Number**: Sequential numbering
- **Duration**: Timestamp and length
- **Scene Title**: Descriptive name
- **Purpose**: What this scene accomplishes

### Visual Elements
- **Shot Type**: Wide, medium, close-up, screen recording, etc.
- **Screen Composition**: Layout of visual elements
- **Camera Movement**: Pans, zooms, transitions
- **On-Screen Elements**: Text, graphics, code, UI elements
- **Color Palette**: Specific colors for branding/emphasis

### Animation Details
- **Entry Animations**: How elements appear
- **Movement**: Dynamic elements during scene
- **Exit Animations**: How scene transitions
- **Timing**: Precise timing for animations

### Audio Integration
- **Narration Sync**: How visuals sync with voice-over
- **Sound Effects**: UI clicks, transitions, emphasis
- **Music**: Background music cues
- **Pacing**: Speed of visual changes

## Storyboard Elements Legend

### Shot Types

| Type | Description | Use Case |
|------|-------------|----------|
| **Wide Shot (WS)** | Full screen view | Establish context, show complete interface |
| **Medium Shot (MS)** | Focused area of screen | Highlight specific section or workflow |
| **Close-Up (CU)** | Detailed view | Emphasize specific element or text |
| **Extreme Close-Up (ECU)** | Very tight on detail | Show exact mouse position, button click |
| **Screen Recording (SR)** | Live capture | Demonstrate actual workflow |
| **Animation (ANIM)** | Illustrated/animated | Explain concepts, show data flow |

### Transition Types

| Transition | Description | Use Case |
|------------|-------------|----------|
| **Cut** | Instant change | Quick scene changes, maintain energy |
| **Fade** | Gradual transition | Scene endings, topic changes |
| **Dissolve** | Cross-fade | Related scenes, smooth progression |
| **Wipe** | Directional transition | Moving between interface sections |
| **Zoom** | Scale transition | Focus in/out on details |
| **Slide** | Lateral movement | Moving through workflow steps |

### Animation Styles

| Style | Description | Use Case |
|-------|-------------|----------|
| **Fade In/Out** | Opacity change | Gentle introduction of elements |
| **Slide In** | Enter from edge | Introduce panels, callouts |
| **Scale** | Size change | Emphasize elements |
| **Highlight** | Color/glow effect | Draw attention |
| **Typing Effect** | Simulated typing | Code demonstrations |
| **Pulse** | Rhythmic scaling | Call attention repeatedly |
| **Path Animation** | Follow line/curve | Show data flow, connections |

### On-Screen Text Types

| Type | Description | Specifications |
|------|-------------|----------------|
| **Title** | Scene/section header | 48pt, bold, brand color |
| **Subtitle** | Supporting text | 32pt, regular, white |
| **Callout** | Highlight info | 24pt, box with arrow |
| **Code** | Programming text | Monospace, syntax highlighted |
| **Label** | UI element name | 18pt, semi-transparent background |
| **Caption** | Additional context | 16pt, bottom screen |

## Production Guidelines

### Screen Recording Settings
- **Resolution**: 1920x1080 (1080p)
- **Frame Rate**: 30fps or 60fps
- **Aspect Ratio**: 16:9
- **Cursor**: Enhanced visibility (ring or highlight)
- **Window Size**: Consistent across recordings

### Animation Timing
- **Text Fade In**: 0.3-0.5 seconds
- **Slide Transitions**: 0.4-0.6 seconds
- **Emphasis Pulse**: 0.2 seconds (repeat 2-3x)
- **Code Typing**: 50-80 characters per second
- **Wait After Action**: 0.5-1.0 seconds (let viewer process)

### Callout Guidelines
- **Arrow Style**: Curved with subtle drop shadow
- **Box Style**: Rounded corners (8px radius)
- **Background**: Semi-transparent with 80% opacity
- **Position**: Avoid covering important elements
- **Duration**: 3-5 seconds minimum (reading time)

### Motion Graphics
- **Easing**: Ease-in-out for natural movement
- **Duration**: 0.5-1.0 seconds for most animations
- **Complexity**: Keep animations simple and purposeful
- **Consistency**: Use same animation style throughout series

## Storyboard Review Process

### Pre-Production Review
1. **Content Accuracy**: Technical reviewers verify information
2. **Visual Clarity**: Ensure shots communicate effectively
3. **Timing**: Validate duration estimates
4. **Brand Compliance**: Check colors, fonts, style
5. **Accessibility**: Ensure readability and clarity

### Production Alignment
1. **Shot List Creation**: Convert storyboard to production checklist
2. **Asset Preparation**: Gather graphics, code samples, data
3. **Screen Recording Plan**: Identify what needs to be recorded
4. **Animation Specification**: Detailed motion graphics brief

### Post-Production Reference
1. **Editing Guide**: Use storyboard for editing decisions
2. **Quality Check**: Compare final video to storyboard
3. **Deviation Documentation**: Note and approve changes
4. **Iteration**: Update storyboard based on learnings

## Accessibility Considerations

### Visual Accessibility
- **Contrast Ratio**: Minimum 4.5:1 for text
- **Font Size**: Minimum 18pt for on-screen text
- **Color Independence**: Don't rely solely on color
- **Motion**: Provide static alternatives for animations

### Cognitive Accessibility
- **Pacing**: Allow time to read and process
- **Complexity**: Introduce concepts gradually
- **Consistency**: Use familiar patterns
- **Clarity**: Avoid visual clutter
