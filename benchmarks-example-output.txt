Run benchmarks and generate performance reports

Usage: cost-ops run [OPTIONS]

Options:
  -o, --output <OUTPUT>    Output directory for benchmark results [default: benchmarks/output]
      --no-summary         Skip generating summary markdown report
  -f, --filter <FILTER>    Benchmark filter (e.g., "cost_calculation", "aggregation")
  -h, --help               Print help
